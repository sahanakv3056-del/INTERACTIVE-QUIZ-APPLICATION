DESCRIPTION:
🔹 What this code does
It creates a Quiz App where:
Questions are shown one by one
User selects an answer
Score is calculated
Timer runs
Final result is displayed

🔹 Main Parts
1. quizData
👉 Stores all questions, options, and correct answers
JavaScript
question, choices, correct
2. Variables
👉 Keep track of quiz state
currentQuestion → which question you’re on
score → correct answers count
timeLeft → timer value
3. startQuiz()
👉 Starts the quiz
Resets score & timer
Shows first question
Starts timer
4. showQuestion()
👉 Displays current question and options
Creates buttons for each choice
5. selectAnswer()
👉 Runs when user clicks an option
Checks if answer is correct
Updates score
Highlights correct/wrong answer
Shows Next button
6. nextQuestion()
👉 Moves to next question
If no more → ends quiz
7. startTimer()
👉 Runs countdown every second
Ends quiz when time = 0
8. endQuiz()
👉 Shows final score
Hides quiz
Shows restart button
9. Restart Button
👉 Resets everything and starts again
